# Мариупольский городской чат V2.31

Воссоздание легендарного [мариупольского городского чата](https://web.archive.org/web/20051211171029/http://chat.mariupol.com.ua/) — по дизайну и духу оригинала 2005 года.

## Запуск

```bash
npm install
npm start
```

Откройте [http://localhost:3000](http://localhost:3000)

## Что внутри

- **Главная страница** — точная копия входа в чат с архива Wayback Machine (декабрь 2005)
- **Каналы**: Городской сад, Квартал свиданий, Неадминистрируемый, Карточный домик, (*) Закрытый канал
- **Смайлики** — все оригинальные GIF из архива (`:)`, `:ПИВО:`, `:ЧМОК:`, `:лол:` и др.)
- **WebSocket-чат** — сообщения в реальном времени, список чатлан онлайн
- **Новости, модераторы, рейтинг** — из архива 2005 года

## Стек

- Node.js + Express — сервер
- ws — WebSocket для чата
- Статический HTML/CSS в стиле 2005

## Структура

```
public/
  index.html      — вход в чат
  chat.html       — комната чата
  css/            — оригинальные стили
  i/smile/        — смайлики из архива
  i/name.gif      — логотип «Мариуполь»
server/
  index.js        — сервер + WebSocket
```

## Разработка

```bash
npm run dev   # с автоперезагрузкой (--watch)
```

## Деплой

### Render (рекомендуется)

1. Форкните/залейте репозиторий на GitHub
2. [render.com](https://render.com) → **New** → **Blueprint** → подключите репо
3. Render подхватит `render.yaml` автоматически

### Docker

```bash
docker build -t chat-mariupol .
docker run -p 3000:3000 chat-mariupol
```
